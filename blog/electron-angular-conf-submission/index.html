Abstract

With the introduction of Github's Electron it is now possible to build Mac, Windows, and Linux desktop applications using Web-standard technologies. Now we can leverage the our HTML/CSS/Javascript skills to create cross-platform applications with Node.js integrated into a Chromium. Join us as we explore setting up, building and deploying a desktop app using Electron and Angular 2.

Description

Last year, Github released Electron, the project that their Atom application is based upon. Now companies like Slack, Microsoft, Ionic and Wordpress are using Electron to build applications and extensions for their Web of mobile applications for the desktop.

At its heart Electron is an integration of Node.js and Chromium, Google's open-source browser project. That means you can program applications for Mac, Windows, and Linux using Javascript. And you can build your user interface using HTML/CSS/Javascript. Pick which ever framework flavor you'd like and build your UI just like you would build your Web applications.

Angular 2, the latest versions of Google's popular MV* framework, assembles TypeScript and RxJS into a powerful modern framework.

Join us as we combine these powerful forces and give you an overview of how you can get started building desktop application using Electron and Angular.

Abstract

Ionic 2 has been released and while key concepts remain, your Ionic 1 experience may leave you a bit lost when addressing the new version. Join us as we take an application through the transition from Ionic 1 to Ionic 2.

Description

[Please Note: Ionic 2 (and Angular 2) have not been released but by the time of this conference it will be. Or at least, if it hasn't been released then something horrible has occurred and you'll cancel this important and amazing talk for something reality-based.]

Ionic 2 is a framework that allows Web-standards developers to assemble iOS, Android and Windows mobile applications. Ionic is a stack of three technologies:

  1. A set of CSS-based platform style-compliant components used to build the mobile UI.

  2. Angular 2, the popular Javascript framework which leverages TypeScript, RxJS and other technologies to build your mobile application.

  3. Apache Cordova a framework with plugins that facilitate communication with features of the device like the Camera and Geolocation.

There are thousands of Ionic 1 applications already in the app stores, but there are compelling reasons to update your code to Ionic 2: speed, maintainability, and speed. Did we mention speed?

Join us as we take a perfectly innocent Ionic 1 app and tear its heart out and place it into an Ionic 2 where it will work oh so much better.
