Summary

Leif Wells is a web, mobile and desktop application developer living in Atlanta, Georgia. Having spent decades in development, Leif provides insight into his experiences at his blog Leif's Development Blog-o-rama as well as speaking at user groups and conferences.

Biography

Leif Wells is a web, mobile and desktop application developer living in Atlanta, Georgia. Over the years Leif has created applications using products using Director, Flash and Flex. These days he is focused on using Web-standard technologies (HTML/CSS/Javascript). Leif has recently built projects using Appcelerator Titanium, Angular, Ionic and React. An early adopter, Leif started using Angular 2 and Ionic 2 in early 2016.

Leif provides insight into his experiences at his blog, Leif's Development Blog-o-rama as well as speaking at user groups and conferences.
